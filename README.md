# DTF 讀書會

# Setup
Follow the guide [here](https://bootyburglar.medium.com/marie-kondo-your-python-dev-environment-391485be9b3f) to setup your python virtual environment. 

```bash
$ mkvirtualenv -a dtf
```
before starting any projects, use the `workon` command
```bash
$ workon dtf
```
